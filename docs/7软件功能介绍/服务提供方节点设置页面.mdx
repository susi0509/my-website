服务提供方节点设置页面中可以为服务提供方节点添加一系列的配置属性，并进行相应的修改操作。

### Provider 标题

设置服务提供方节点的标题。

### Provider Name

设置服务提供方节点的名称。

### Provider instanceidDefined

设置服务提供方节点的默认实例id名称。

### 描述

对服务提供方节点添加描述。

### Provider relativeUri

选择该Provider是否使用ProviderrelativeUri。

### 删除

删除服务提供方节点。

### 更新

更新服务提供方节点的设置。
