点击创建新项目后，软件将会跳转到创建新的适配器模板导航页面，在该页面中继续点击“下一步”按钮后，将会跳转到创建适配器模板页面中。

### 项目名称

命名该适配器项目的项目名称。

### 适配器ID

为该适配器创建一个全局唯一的ID。该ID为Datalinks适配器模板生成软件自动生成并锁定的ID。

### 适配器名称

为创建的适配器命名。

### 服务Domain

为该适配器定义服务uri

### OSLC服务Domain Alias

为该适配器创建服务域uri的别名。

### Java NS Base

为该适配器创建工作空间目录结构。

### 端口

为该适配器设置服务端口。

### 项目属性

选择是否使用根节点Root Service。

### 其他设置

选择是否创建完成后编译。

### 上一步

返回上一个页面。

### 确定

确定根据以上填写的配置信息，创建新的适配器。
