在点击适配器创建按钮，并依据导航，点击“确定”按钮后，将会生成适配器模板。通过“确定”按钮的点击跳转，将会进入到服务配置页面，其中，节点的设置页面如图所示。

### Catalog标题

设置节点的标题。

### Catalog Name

设置目录节点的名称。

### Catalog instanceidDefined

设置目录节点默认单例名称。

### 描述

给目录节点添加描述信息。

### Catalog relativeUri

选择Catalog是否使用Catalog relativeUri。

### 确定

确定对适配器模板的节点设置。
