- 点击资源条目进行资源的配置。

![](/img/tutorial/image-20220111142112697.png)

​	图 54 资源配置

资源是适配器服务所操作的数据，主要用于针对不同的业务需求所解析的数据。按照具体的业务需求，资源具有不同的颗粒度，比如，对于同一个Excel文件，如根据业务的需求可以以一行为一个单位，那资源的定义就可以为行；若业务的需求比较精细，也可以将该文件中的某一单元格作为一个资源进行定义。

- 在点击资源选项卡后，将会跳转出资源树状条目信息栏。

- 鼠标右键“资源”选择添加资源。

![](/img/tutorial/image-20220111142154349.png)

​	图 55 树状节点目录

- 点击Default Resource Content树状节点目录，为其添加相关配置项。

![](/img/tutorial/image-20220111142225350.png)

图 56 配置表单项

- 在Default Resource Content节点目录表单中填写入相应的配置内容。

填写示例：

​																														表 8 填写示例

| Resource标题 | Excel Content Resource Shape |
| :----------: | :--------------------------: |
| ResourceName |         ExcelContent         |

![](/img/tutorial/image-20220111142312670.png)

​	图 57 表单填写

- 点击“添加属性”按钮。

![](/img/tutorial/image-20220111142343780.png)

​	图 58 添加属性

- 页面将会跳转出属性配置表单页面。

- 在该页面中进行相关属性的配置。

![](/img/tutorial/image-20220111142423150.png)

​	图 59 属性配置

- 在添加Resource属性表单页面中填写入相应的配置内容。

填写示例：

​																														表 9 填写示例

|   属性标题   |             columnId             |
| :----------: | :------------------------------: |
|   属性名称   |             columnId             |
|     描述     | Description for property content |
|   属性类型   |         REF##XMLLiteral          |
| 属性出现次数 |            ExactlyOne            |
|     只读     |               不选               |
|   属性定义   |             columnId             |

![](/img/tutorial/image-20220111142528043.png)

​	图 60 表单填写

- 在属性表单填写完毕以后，点击更新按钮，将会对已配置好的属性进行保存。

- 已配置好的属性将会在资源配置页面如图进行显示。

![](/img/tutorial/image-20220111142602567.png)

![](/img/tutorial/image-20220111142612022.png)

​	图 61 添加属性

- 此时返回Service资源配置节点，点击“请选择资源”按钮，将会看到刚才已配置好的资源。

![](/img/tutorial/image-20220111142640133.png)

​	图 62 属性配置

- 返回Service资源配置节点，此时连接刚才配置的节点，并点击更新，页面将会提示，更新节点信息成功，表示服务与信息的连接已完成。

![](/img/tutorial/image-20220111142704999.png)

图 63 服务与资源连接
