本使用说明书为方便用户阅览，以新建一个需求适配器Excel为例，进行适配器模板创建的演示。其他适配器的模板创建工作同理。
当选择好工作空间，点击“确定”按钮后，软件页面将会跳转到创建新项目页面。
<center>

![](/img/tutorial/image-20220111140954339.png)

</center>
<center>
图 37 创建新项目页面
</center>
在该页面中选择创建新项目按钮，软件页面将会跳转到一个新的适配器创建向导页面中。
<center>

![](/img/tutorial/image-20220111141009426.png)

</center>
<center>
​图 38 创建新的适配器模板项目
</center>
在该页面中继续点击创建向导，点击“下一步”后，将会在软件页面中跳转出创建新项目填写配置信息页面。
<center>

![](/img/tutorial/image-20220111141021556.png)

</center>
<center>
​图 39 填写项目配置信息
</center>
在该页面中，需要用户填写项目名称，适配器名称，OSLC服务Domain，OSLC服务Domain Alias，Java NS Base以及选择是否勾选其他设置选项等几项内容。

在此，用户手册操作演示示例数据如下所示：

表 3 数据演示

<center>


|       项目名称       |           ExcelAdapter           |
| :------------------: | :------------------------------: |
|      适配器名称      |           ExcelAdapter           |
|    OSLC服务Domain    | http://oslcadapter.com/ns/Excel# |
| OSLC服务Domain Alias |        oslcadapter-sample        |
|     Java NS Base     |        com.sample.adapter        |
|         端口         |               9292               |

</center>

在软件中该页面的除以上几项的其他选项栏，配置信息已自动生成或系统自行提供，在此处不可修改。在使用该软件自动生成适配器模板时，直接使用软件提供的默认值即可。

在软件的相应位置填写好如上信息后，点击该页面中的确认按钮，进入下一步的操作。

<center>

![](/img/tutorial/image-20220111141110119.png)

</center>
<center>
图 40 填写配置信息
</center>
<center>

![](/img/tutorial/image-20220111141121428.png)

</center>
<center>
图 41 模板创建成功
</center>
当成功填写进信息后，软件前端将会显示成功创建模板，并在新跳转的页面中显示目前所建的目录层级结构。
