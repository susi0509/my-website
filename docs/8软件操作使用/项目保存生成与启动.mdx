- 点击文件选项卡，点击保存项目，对已构建的项目进行保存。

![](/img/tutorial/image-20220111142743941.png)

图 64 保存项目

- 保存成功后，点击项目选项卡，构建，生成Java项目选项，对项目进行java代码框架生成。

![](/img/tutorial/image-20220111142817573.png)

​	图 65 生成代码框架

生成的适配器框架代码包结构说明如下表：

​																													表 10 代码结构

|     包      |            说明             |         备注         |      |
| :---------: | :-------------------------: | :------------------: | ---- |
|  generated  |            Libs             |    存放OSLCjar包     |      |
|     Src     |           Configs           | 存放适配器项目配置类 |      |
| controllers |    存放适配器项目控制类     |                      |      |
|  resolvers  |    存放适配器项目解析类     |                      |      |
|  resources  |    存放适配器项目资源类     |                      |      |
|     Src     | 存放Datalinks生成的json文件 |                      |      |

- 打开代码编辑工具，导入刚才已生成好的java项目。

![](/img/tutorial/image-20220111142917357.png)

图 66 项目导入

- 导入成功后，运行该项目，由此适配器代码框架已生成成功。

![](/img/tutorial/image-20220111143108242.png)

图 67 生成代码框架
